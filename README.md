# dr-component
基于 [element-ui](https://element.eleme.cn/) 封装的业务组件库
# Usage
```
npm i element-ui dr-component
```

**example:**

*.vue
```html
// html
<template>
  <PageTable />
</template>
```

```javascript
// script
import { PageTable } from 'dr-component'
export default {
  components: {
    PageTable
  }
}

```
